Projects

YouSure?

In BitCamp 2024, I participated in a hackathon with a few friends and we worked on a website that takes you health insurances and compares it and tells you which is better? Sounds too familiar? We added more! We made a chatbot where you could ask the app to quickly summarize your insurance, highlight anything important which it would recognised as we have trained it to do so!The front end, built with React, offers a user-friendly interface for details, file uploads, and comparisons. The Flask backend supports user login, insurance analysis, and comparisons. Using user data, we predict associated risk with an XGBoost model. PDFs are parsed and stored in a vector database (Pinecone), leveraging RAG with LangChain for tailored results. GPT-3.5 Turbo enhances search functionality, ensuring accurate and up-to-date results.
